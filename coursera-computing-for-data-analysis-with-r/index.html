Coursera - Computing for Data Analysis with R

I recently completed Computing for Data Analysis with
R. It was basically an
introduction to R.

Why learn R when one can analyze data with Python and
pandas?
For starters, I want to understand how and when to use statistical methods to gain insight
from data. A lot of the online courses that teach on these methods use R, so I figured it would be
useful to understand the syntax of R so that I could more easily follow
along. Also, learning a new programming language usually introduces me
to a concept that I didn't fully understand or get to play with in
whatever language I was using most at the time.

I was introduced to the *apply methods in R (apply, lapply, etc),
which were really useful. They seem pretty similar to list
comprehensions in Python.

ggplot2 seems pretty awesome. Actually, being introduced to the concept of
'grammar of graphics' alone was really cool. (Hey look Greg
Lamp
wrote ggplot for
Python)

The video lectures were great, and I was pleased to discover that
it's possible to watch the lectures at 1.5x normal speed. This was
revolutionary for me.

The class was only 4 weeks long, so I didn't dive very deep into R.
Maybe I'll find use for it in the future, like if I take Data
Analysis on Coursera.
Or maybe I'll stick with Python for data analysis.
