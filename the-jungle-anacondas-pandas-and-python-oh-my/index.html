The Jungle (anacondas, pandas, and python - oh my!)

Want to use Python to analyze data? Install Anaconda, IPython Notebook,
and pandas. Tips and tricks below.

Motivation

If we want to get our hands dirty with data in Python, we need to
install a handful of useful packages like
NumPy, pandas, matplotlib, IPython, etc. We can either try to install all of those ourselves and hope that
the latest release of each package is compatible with all of the others,
or we can just go download
Anaconda from ContinuumIO.

Installation

Installation on a Mac is super simple. Note that it takes a long time to download because you get all
of these great packages.

IPython Notebook

Let's launch Anaconda. On my Mac, this is as simple as opening the
Anaconda Launcher shortcut in my dock. From the Anaconda launcher, open
IPython Notebook.

If you've yet to experience IPython Notebook, then you are in for a
treat. This is the best way to do exploratory data analysis.

The IPython Notebook is a web-based interactive computational environment where you can combine code execution, text, mathematics, plots and rich media into a single document

You'll understand how awesome it is within a couple of minutes if you
just go to the IPython Notebook
website, scroll down a bit, and watch their explanatory YouTube
video.

Slightly off-topic: MATLAB and PowerPoint

If only I had known about IPython Notebook (and numpy and matplotlib, etc)
when I was carrying out research during my MS at UIUC.
Instead I wrote MATLAB scripts, saved the resulting plots to one huge PowerPoint
file, and then chose the best plots to save in a smaller
PowerPoint that I would email to my advising professor.
e'd look at PowerPoint, annotate, and send back to me. Then I'd tweak my scripts until the plots
looked like what he drew in PowerPoint, save to a new PowerPoint, and
resend. If prompted, I'd also send the MATLAB scripts as another
attachment with the PowerPoint file.

Imagine if I could have just saved the code and
plots in an IPython Notebook. For small edits, my professor could have just tweaked the
code in the IPython Notebook directly instead of trying to draw his
changes on the image in
PowerPoint. That's also much better than opening up MATLAB, tweaking the MATLAB code, saving it as a new
MATLAB file, and emailing back. He would actually be able to easily read
and edit on a laptop (opening MATLAB and PowerPoint at the same time is best with two
screens + a powerful processor). And I wouldn't have had the overhead of
dealing with so many files (script-01.14.m, plots-01.14.pptx,
script-01.14.revised.m, plots-01.14.revised.m, script-01.15.m, etc).

Inline Plots

If we start a new notebook and plot something

from pylab import *

X = np.linspace(-np.pi, np.pi, 256,endpoint=True)
C,S = np.cos(X), np.sin(X)

plot(X,C)
plot(X,S)

show()

the plot will pop up in a separate window:


I'd rather have the plot embedded into the IPython notebook. Fortunately, inline plots are supported by IPython and easy to enable.

To enable inline plots on a Mac, we need to edit /anaconda/bin/ipython_mac.command
so the Anaconda launcher knows to start IPython notebook in
inline mode. Just open up /anaconda/bin/ipython_mac.command and replace $DIR/ipython notebook with $DIR/ipython notebook --pylab
inline.

After saving your changes to /anaconda/bin/ipython_mac.command,
relaunch IPython Notebook from the Anaconda launcher. Now when you run
some plotting code, you'll get an inline plot!

pandas

http://pandas.pydata.org/ is a powerful Python data analysis library. In the next post we'll play around with pandas. For now let's just make
sure pandas is accessible and working by trying some demo code from
pandas' documentation.

import pandas as pd
import numpy as np
dates = pd.date_range('20130101',periods=6)
print pd.DataFrame(np.random.randn(6,4),index=dates,columns=list('ABCD'))
